Join the Server

Follow these instructions to join our cracked Minecraft server:

  1. Open Minecraft and go to the Multiplayer menu.
  2. Click on "Add Server".
  3. Enter the server name as "Somewhere We Know".
  4. Set the server address to basis-pixel.joinmc.link and the port to 10983.
  5. Save and click "Join Server".

We look forward to seeing you in-game!
